# 付録

**目的**: 読者のための参照資料を提供すること。

本編の旅を補完するための重要な資料群です。専門用語の定義、詳細な実行トレース、そしてRemlが依存する巨人の肩（サードパーティライブラリ）についてまとめています。

コードを読み解く中で迷ったとき、あるいは特定のトピックについて手っ取り早く情報を得たいとき、この付録が役に立つはずです。

## 本部の構成

- **A: 用語集と索引**
  - Reml独自の用語やコンパイラ用語の定義、およびモジュール名と機能の対応表を提供します。
- **B: エンドツーエンド実行トレース**
  - 具体的なソースコードがコンパイルされ実行されるまでの全工程を、実際の関数呼び出しレベルで追跡した記録です。
- **C: サードパーティライブラリ一覧**
  - Remlの実装において重要な役割を果たしている外部クレート（ライブラリ）の目的と選定理由をリスト化します。
